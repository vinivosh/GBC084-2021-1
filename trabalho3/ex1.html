JavaScript

O JavaScript é uma linguagem de programação que, junto com HTML e CSS formam as três teconologias principais da internet. Pode-se dizer que o JS é o que verdadeiramente deixa as páginas web interativas, enquanto o HTML se preocupa com o esqueleto das páginas e o CSS com seu visual.
